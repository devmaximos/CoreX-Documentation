# /giveaway_reroll Command Documentation

---

## Overview

The `/giveaway_reroll` command allows authorized users to reroll an existing giveaway and select new winners. This can be used if the original winners cannot claim their prize or if you want to randomly pick new winners.

**Key Features:**
- Reroll an active or ended giveaway.
- Requires only the giveaway message ID or link.

**Permissions:**
- **User:** Must have permission to manage giveaways (usually Admin/Moderators).  
- **Bot:** Must have `Send Messages` permission in the channel containing the giveaway.

---

## Command Syntax

```bash
/giveaway_reroll message:<string>
